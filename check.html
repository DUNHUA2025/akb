🚀 後端部署方式（免費）

    


      

        
✅ 方法一：Railway（推薦，最簡單）

        
    
          
  1. 前往 railway.app，用 GitHub 帳號登入
    2. 
          
  2. 點「New Project」→「Deploy from GitHub repo」
    3. 
          
  3. 選擇 DUNHUA2025/akb repo
    4. 
          
  4. Root Directory 設為 server
    5. 
          
  5. 部署完成後取得 URL（如 https://akb-xxx.railway.app）
    6. 
          
  6. 在各 HTML 頁面 <head> 最頂部加入：
    
            <script>window.AKB_API_URL='https://akb-xxx.railway.app';</script>
          
    7. 
        

      


      

        
方法二：Render

        
    
          
  1. 前往 render.com，用 GitHub 登入
    2. 
          
  2. New → Web Service → 選 DUNHUA2025/akb
    3. 
          
  3. Root Directory: server，Build Command: npm install，Start Command: node server.js
    4. 
          
  4. 取得 URL 後同上設定 window.AKB_API_URL
    5. 
        

      


      

        
方法三：本機測試

        
    
          
  1. 電腦需安裝 Node.js（下載）
    2. 
          
  2. 終端機執行：cd akb/server && npm install && npm start
    3. 
          
  3. 在此頁輸入 http://localhost:3001 即可測試
    4.
